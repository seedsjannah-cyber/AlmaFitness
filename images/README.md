## Quick Start

### Option 1: Use Your Own Images
Add your professional photos following the specifications above.

### Option 2: Use Placeholder Images Temporarily
Download free images from:
- [Unsplash](https://unsplash.com/s/photos/fitness)
- [Pexels](https://www.pexels.com/search/wellness/)
- [Pixabay](https://pixabay.com/images/search/fitness/)

### Option 3: Use Temporary URLs
The website will work with placeholder URLs while you gather real images.

## Image Optimization

Before adding images, optimize them using:
- [TinyPNG](https://tinypng.com/)
- [Squoosh](https://squoosh.app/)

Target file sizes:
- Hero background: < 500KB
- Portrait: < 300KB
- Testimonials: < 100KB each
- Products: < 200KB each
